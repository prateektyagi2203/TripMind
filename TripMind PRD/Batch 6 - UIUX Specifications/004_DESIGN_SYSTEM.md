# Design System

Version: 1.0

Status: UI/UX Specification

------------------------------------------------------------------------

# Purpose

Define the visual language, reusable components and interaction rules
for TripMind. The design system ensures a consistent experience across
Android and iPhone while allowing platform-specific adaptations.

------------------------------------------------------------------------

# Design Principles

-   Simple over decorative
-   Travel information first
-   AI assistance should feel natural
-   Large touch targets
-   Consistent spacing
-   Accessible by default

------------------------------------------------------------------------
