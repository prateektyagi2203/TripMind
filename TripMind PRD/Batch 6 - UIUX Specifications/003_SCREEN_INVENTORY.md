Version: 1.0

# Purpose

This document defines every screen planned for TripMind MVP and future
releases. It serves as the master index for UX design, Flutter
implementation and navigation.

# Design Principles

-   One primary purpose per screen
-   Consistent navigation
-   Minimal cognitive load
-   Offline-aware
-   AI-assisted wherever appropriate
