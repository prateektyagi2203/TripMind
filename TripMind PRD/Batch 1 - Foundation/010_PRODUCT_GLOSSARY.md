# Product Glossary

Version: 1.0

Status: Foundation

------------------------------------------------------------------------

# Purpose

This glossary provides concise definitions of the primary business,
technical and AI concepts used throughout the TripMind repository. It
complements `009_TERMINOLOGY.md` and serves as a quick reference for
developers, designers and AI coding assistants.

------------------------------------------------------------------------

# Business Terms

  -----------------------------------------------------------------------
  Term                    Definition
  ----------------------- -----------------------------------------------
  Trip                    The primary entity representing a complete
                          journey.

  Traveler                A registered TripMind user.

  Family                  A group of travelers sharing a trip.

  Destination             Country, city or region being visited.

  Activity                Planned sightseeing, booking or experience.

  Booking                 Reservation for hotel, activity or transport.

  Itinerary               Chronological plan of a trip.

  Memory                  AI-generated collection of photos, places and
                          journal entries.
  -----------------------------------------------------------------------

------------------------------------------------------------------------

# AI Terms

  -----------------------------------------------------------------------
  Term                    Definition
  ----------------------- -----------------------------------------------
  TripMind Brain          Central intelligence responsible for context
                          and recommendations.

  Trip Context            Live contextual object used by every engine.

  AI Concierge            Conversational travel assistant.

  Recommendation          AI-generated suggestion based on context.

  Travel DNA              Long-term profile describing user travel
                          preferences.

  Context Engine          Maintains and updates Trip Context.

  Memory Engine           Builds searchable travel memories.
  -----------------------------------------------------------------------

------------------------------------------------------------------------

# Technical Terms

  -----------------------------------------------------------------------
  Term                    Definition
  ----------------------- -----------------------------------------------
  Engine                  Independent business capability such as Flight
                          Engine.

  Module                  Collection of related engines, UI and services.

  Plugin                  Installable capability extending TripMind.

  Destination Pack        Offline package containing travel intelligence
                          for a destination.

  Provider                Third-party service integration such as Grab or
                          Google Maps.

  Repository              Source code and documentation managed in
                          GitHub.
  -----------------------------------------------------------------------

------------------------------------------------------------------------

# Offline Terms

  -----------------------------------------------------------------------
  Term                    Definition
  ----------------------- -----------------------------------------------
  Offline First           Core functionality works without network
                          connectivity.

  Local Cache             Device storage used for fast access.

  Sync Engine             Synchronizes local data with cloud services.

  Conflict Resolution     Strategy for merging offline and online
                          changes.
  -----------------------------------------------------------------------
