# 014_MOBILE_SECURITY

## Purpose

Define mobile application security requirements.

## Device Security

-   Secure storage
-   Biometric authentication
-   Screen lock support
-   Clipboard protection for sensitive data

## Application Security

-   Certificate pinning (future)
-   Root/jailbreak detection (future)
-   Obfuscation
-   Integrity checks

## Data Protection

-   Encrypted local database
-   Secure token storage
-   Automatic session expiry

## Acceptance

Mobile clients protect sensitive data at rest and in transit.
