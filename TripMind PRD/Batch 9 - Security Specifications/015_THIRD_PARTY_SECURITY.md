# 015_THIRD_PARTY_SECURITY

## Purpose

Security requirements for external integrations.

## Requirements

-   HTTPS only
-   API key management
-   Least privilege
-   Vendor risk assessment
-   Rate limiting
-   Timeout handling

## Monitoring

-   Availability
-   Error rates
-   Credential rotation
-   Audit logs

## Acceptance

All third-party integrations follow approved security standards.
