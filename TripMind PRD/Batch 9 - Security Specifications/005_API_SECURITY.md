# 005_API_SECURITY

## Purpose

Protect all backend APIs.

### Controls

-   JWT validation
-   HTTPS only
-   Input validation
-   Output encoding
-   Rate limiting
-   Idempotency where applicable

### Headers

-   HSTS
-   CSP (web)
-   X-Content-Type-Options

### Acceptance

-   OWASP API Top 10 mitigations implemented.
