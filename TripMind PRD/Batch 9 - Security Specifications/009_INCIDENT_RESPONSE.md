# 009_INCIDENT_RESPONSE

## Purpose

Define the incident response process.

### Phases

1.  Detection
2.  Triage
3.  Containment
4.  Eradication
5.  Recovery
6.  Post-incident review

### Severity

-   Critical
-   High
-   Medium
-   Low

### Requirements

-   Incident owner
-   Communication plan
-   Evidence preservation
-   Root cause analysis

## Acceptance

-   Security incidents documented and reviewed.
